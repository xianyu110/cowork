# Cowork Landing Page

一个现代化的 Cowork 产品落地页和价格页面，展示 Claude AI 的 Cowork 功能特性。

## 关于 Cowork

Cowork 是 Claude 推出的革命性功能，将 Claude Code 的执行能力带到桌面端，专为知识工作而设计。它不仅仅是对话，而是能在你的实际工作环境中完成复杂任务的智能助手。

### 核心特性

- **📁 在文件环境中工作** - 在本地桌面运行，直接在文件夹中读取、整理和创建文件，而不仅仅是生成新内容
- **🤖 启动子代理** - 将复杂任务分解给独立的子代理并行处理，每个都有全新的上下文，突破单线程限制
- **⏱️ 交付后查看** - 处理任务编排、上下文管理和进度跟踪，你可以随时查看进展或让它自主运行
- **🔗 MCP 连接器** - 通过 MCP 连接器链接到实际工作环境，访问邮件、日历、文档等
- **🌐 Chrome 集成** - 执行浏览器任务：读取网页、填写表单、提取数据、跨标签页导航

### 何时使用 Cowork

**使用 Cowork 的场景：**
- 工作涉及本地文件操作（整理、分析、创建输出）
- 任务涉及大量材料或多个数据源
- 工作受益于并行探索
- 想要委托任务并回来查看结果
- 任务可能超出普通对话的时长

**使用普通对话的场景：**
- 需要实时互动协作
- 任务更快速、更像对话
- 想要全程参与
- 探索性工作，与 Claude 一起思考

### 实际应用示例

**访问本地文件：**
- "整理我的下载文件夹，按类型和项目分组"
- "找出可以删除的重复照片和截图"

**创建深度工作成果：**
- "我要去里斯本一周。搜索邮箱和日历获取行程详情，搜索天气、社区、餐厅和景点，创建行程文档"
- "我在 Teams 和 Zendesk 上有 50+ 条客户反馈。找出主要投诉，然后搜索 SharePoint 中的文档开始起草回应"

**完成长时间运行的任务：**
- "浏览整个笔记库，找出孤立的笔记、错过的联系，建议更好的文件夹结构"
- "阅读 /contracts 中的所有文档，为每个创建关键条款、续约日期和义务的摘要"

**并行处理复杂工作：**
- "我在评估四个供应商。启动子代理研究每个的定价、支持声誉和集成选项，给我对比"
- "用子代理从三个角度分析这个决定：财务影响、客户体验和运营风险，综合推荐"

**异步工作：**
- "周五有绩效评估。搜索 Slack、Google Drive 和 Asana 查看完成的任务、项目更新、同事反馈，起草会议准备表"

## 使用技巧

1. **从提示构建器开始** - 打开 Cowork 时会看到"创建文件"、"处理数据"、"整理文件"、"准备一天"等建议
2. **用完整任务的角度思考** - 描述具有明确终点的任务，如"分析这些文档并创建包含三大主题的幻灯片"
3. **培养直觉** - 从明确界限的任务开始（"整理此文件夹"、"综合这些文档"），逐步扩大到更大项目

## 当前限制

- **仅支持 macOS 桌面端** - 会话不会同步到网页端或移动端
- **无聊天分享** - 会话保留在本地设备上
- **部分功能尚未开放** - Projects、聊天共享、工件共享和 Memory 目前不可用，也不能在对话中途切换模式

本地执行意味着更快的迭代和直接的文件访问，会话保留在设备上并使用机器资源。
